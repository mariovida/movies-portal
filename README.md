# Movies portal

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Swiper.js
- TMDB API

## Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/mariovida/movies-portal.git
   cd movies-portal
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Create an environment file**

   ```bash
   Example is in .env.local.example file.
   ```

4. **Run the development server**

   ```bash
   npm run dev
   ```
